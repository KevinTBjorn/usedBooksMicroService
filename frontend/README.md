# Georgia Tech Library Marketplace Frontend

This is a React TypeScript frontend for the Georgia Tech Library Marketplace system. It provides UI for:

- Adding books
- Searching books
- Placing orders
- Warehouse management (placeholder)

## Pages
- `/addbook` – Add a new book
- `/search` – Search for books
- `/order` – Place an order
- `/warehouse` – Warehouse management

## Setup
1. Run `npm install` in the `frontend` folder.
2. Run `npm start` to launch the development server.
3. The app will connect to backend services via REST API endpoints.

## Dependencies
- React
- TypeScript
- Material UI
- React Router

## API Endpoints
Update the API URLs in each page to match your backend service addresses if needed.
